# τροφή (Trophē) — Precision Nutrition Coaching Platform

## Problem
Professional nutritionists like Michael Kavdas (PN1 certified) manage clients using spreadsheets and generic tracking apps. No platform enforces Precision Nutrition's habit-based methodology at scale.

## Users
- **Coaches**: Professional nutritionists who manage multiple clients
- **Clients**: People seeking guided nutrition coaching with progressive habits

## Core Flows
1. Client signs up → onboarding wizard → body stats → calculated macros
2. Coach assigns first habit → client checks in daily (one tap) → 14-day cycle
3. Coach monitors dashboard → behavioral intelligence → progress/flag clients
4. Client optionally logs food, tracks water, takes supplements
